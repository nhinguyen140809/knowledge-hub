# Knowledge Hub for AI Agents in SDLC

A Knowledge Hub (GraphRAG) service that acts as **long-term memory** and **context provider**
for AI agents across the software development life cycle. It ingests source code, documents and
git history, links them in a **knowledge graph + vector store**, and serves **hybrid retrieval**
over **REST** and **MCP**.

> Project scaffolding (Spring Boot) lands on branch `setup/spring-boot-scaffold` via PR.
